## 문제 풀이 과정
- 문제 분석과 함께 어떤 알고리즘과 자료구조가 적절한지 파악

- 코딩을 통합개발환경 도움 없이 화이트 보드나 종이에 연습
- 시간/공간 복잡도를 고려
- 어떤 테스트 케이스를 통과하는지에 대해 고려함
